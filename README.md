# SIGMA DIGITAL - Sitio Web
### Descripción del Proyecto
SIGMA DIGITAL es un sitio web corporativo para una empresa especializada en desarrollo de software y aplicaciones. La página presenta una interfaz moderna y atractiva que muestra los servicios, proyectos y equipo de la empresa.

El sitio web incluye:


- Página principal con información corporativa**

- Sección "About Us" con presentación del equipo

- Catálogo de servicios ofrecidos

- Galería de proyectos realizados

- Tabla de precios y tarifas

- Sistema de selección de idiomas (Inglés, Español, Euskera)

- Formulario de contacto



### Funcionalidades

- **Diseño Responsivo:** Adaptable a diferentes dispositivos (móviles, tablets, desktop)

- **Selector de Idiomas:** Interfaz disponible en inglés, español y euskera

- **Menú Navegación:** Sistema de navegación intuitivo con efecto hover

- **Tarjetas Interactivas:** Efectos de hover en las tarjetas de equipo y servicios

- **Animaciones CSS:** Fondos animados con partículas flotantes

- **Formulario de Contacto:** Para solicitudes de presupuesto y consultas

- **Sección de Precios:** Tres planes de servicio con características detalladas

# Cómo pueden usarlo los usuarios
### Para Clientes Potenciales:

1.Navegar por las diferentes secciones usando el menú superior

2.Explorar los servicios ofrecidos en la sección "Services"

3.Ver proyectos anteriores en "Projects"

4.Consultar precios en "Prices" y seleccionar el servicio deseado

5.Contactar a la empresa mediante el formulario de contacto

### Para Desarrolladores:

1. Clonar el repositorio

2. Estructura de la pagina:
```
sigma-digital/
│
├── index.html              # Página principal
├── contacto.html           # Formulario de contacto
├── css/
│   ├── style.css           # Estilos principales
│   └── portada.css         # Estilos específicos
├── imagenes/               # Recursos visuales
│   ├── images.jpg          # Logo
│   ├── fondo-bandera-*.jpg # Banderas idiomas
│   └── *.webp              # Fotos equipo
└── README.md               # Documentación
````
3. Personalizar el contenido según necesidades

4. Modificar estilos CSS en los archivos correspondientes

5. Añadir nuevas funcionalidades según requerimientos
